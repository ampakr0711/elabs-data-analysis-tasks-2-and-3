# 📉 Gradient Descent Implementation

The regression model was implemented manually using mathematical optimization techniques.

The model updates weights iteratively to minimize prediction error.

### Linear Regression Equation

```math
y = wx + b
```

### Gradient Descent Update Rule

```math
w := w - α \frac{\partial J}{\partial w}
```
